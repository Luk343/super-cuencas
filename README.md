# super-cuencas
Un plugin funcional para QGIS que automatiza un pipeline hidrológico de 9 pasos, escrito en Python, integrando Whitebox Workflows, GDAL y la API de QGIS, con manejo de CRS, geometrías, archivos temporales, índices morfométricos y visualización, desarrollado para uso real en contexto académico (Escuela de Geografía, UACh).
