# Pipeline de Automatización Hidrológica para QGIS

Este complemento automatiza un flujo de trabajo hidrológico estructurado en 9 etapas consecutivas dentro del entorno QGIS. Diseñado rigurosamente para el análisis geomorfológico y la planificación territorial, el sistema procesa Modelos Digitales de Elevación (MDE) mediante la integración de herramientas avanzadas de teledetección y algoritmos de análisis espacial.

## Características Principales
- **Procesamiento en 9 Pasos:** Desde la corrección de depresiones en el MDE (Depression Filling) hasta la extracción de redes de drenaje e índices morfométricos avanzados.
- **Interoperabilidad de Motores:** Integración nativa de la API de QGIS, abstracción de datos raster/vectorial mediante GDAL y ejecución de algoritmos de alto rendimiento con Whitebox Workflows.
- **Gestión Avanzada de Datos:** Control automatizado de Sistemas de Referencia de Coordenadas (CRS), validación topológica de geometrías y optimización de memoria mediante el uso de archivos temporales.
- **Ámbito de Desarrollo:** Desarrollado bajo estándares profesionales para su aplicación práctica y soporte de investigación en la Escuela de Geografía de la Universidad Austral de Chile (UACh).
